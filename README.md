# 📖 Sensitive Study Pro: Open Source Learning AI
**Ecosistema Nature & Silicium | IA Soberana y Descentralizada**

Sensitive Study Pro es una herramienta de **Onboarding & Learning** automatizado diseñada para profesionales que necesitan procesar grandes volúmenes de documentación técnica, legal o operativa en tiempo récord.

## 🚀 Propósito del Proyecto
Este proyecto nace de la necesidad de eliminar la "fatiga de información". Permite transformar manuales estáticos (PDF), apuntes o fotos de documentos en un entorno de aprendizaje dinámico con evaluación y certificación automática.

## 🛠️ Stack Tecnológico
* **Lenguaje:** Python 3.10+
* **Framework:** [Streamlit](https://streamlit.io/) (Elegido por su robustez en la gestión de estados y procesamiento de datos en tiempo real).
* **Motor de IA:** Integración con **Llama 3** a través de **Groq** (Inferencia de ultra alta velocidad).
* **Procesamiento:** PyPDF2 para extracción de texto (OCR-ready).

## 🔒 Privacidad y Soberanía (BYOK)
Este es un proyecto **"Bring Your Own Key" (Traé tu propia llave)**. 
- No almacenamos datos de los documentos subidos.
- La conexión es directa entre el cliente y el proveedor de IA (Groq).
- Transparencia total en el flujo de datos.
